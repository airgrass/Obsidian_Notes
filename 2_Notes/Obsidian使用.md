## 2 图片处理插件
- Custom Attachment Location:
	- 修改图片保存位置和文件名（不使用其压缩）
		![[Pasted image 20251218235902.jpg]]
- Local Images Plus
	- 用于网络图片本地化
	- 启用自动处理、压缩图片
	- ~~删除孤立的attachment资源（笔记不存在时该命令删不干净）~~
	- 不要修改attachment在笔记中的内容
		![[Pasted image 20251219000238.jpg]]
		![[Pasted image 20251219000322.jpg]]
		![[Pasted image 20251219000623.jpg]]
- Clear Unused Images: 
	- 用于删除未使用的attachments

## 3 让代码块可执行
- Execute Code: 本地执行，不可交互
- Code Emitter: 远程调用，沙盒，可交互
